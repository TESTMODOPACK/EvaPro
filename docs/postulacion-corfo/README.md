> Carpeta de trabajo para preparar la postulación al programa **Start-Up Chile Build** (Corfo). Cofinanciamiento de hasta **CLP $15.000.000** (90% del proyecto) para startups tech-based con menos de 12 meses de inicio de actividades en SII.

## Contenido

| Archivo | Propósito |

| `02-formulario-postulacion-build.md` | Borrador campo por campo del formulario online de Start-Up Chile Build |

- **Razón social**: Ascenda SpA (Chile)
- **Producto**: Eva360 — SaaS B2B de gestión integral del desempeño con IA
- **Antigüedad SII**: < 12 meses ✅ (cumple requisito Build)

- **Estado**: MVP funcional con tenants demo y primeras suscripciones; v3.x en roadmap
- **Founder**: Ricardo Ascenda
